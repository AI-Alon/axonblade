AxonBlade Docs

Everything you need to write, run, and understand AxonBlade programs.

Installation

Requires Python 3.11 or newer. Clone the repo and install from source:

git clone https://github.com/AI-Alon/axonblade.git
cd axonblade
pip install -e .

This installs the ablade CLI command. Verify with:

ablade version
If ablade is not found after install, add the pip scripts directory to your PATH:
export PATH="$(python -m site --user-base)/bin:$PATH"

First Program

Create a file called hello.axb:

>> name = "World"
write(-*cyan*- + "Hello, &{name}!" + -*reset*-)

Run it:

ablade run hello.axb

Running Files

ablade run <file.axb>   # run a program
ablade repl             # start interactive REPL
ablade version          # print version

REPL

Start an interactive session with ablade repl. Each line is evaluated immediately. Multi-line blocks (functions, classes, loops) are entered by ending each line with a continuation — the REPL detects the open block and waits for ECB.

Variables

Use >> to declare a new variable. Re-assign without it.

>> x = 10
x = x + 5   # re-assign
write(x)    # 15
Assigning to an undeclared name raises AxonNameError. Always declare with >> first.

Strings

String literals use double quotes. Embed expressions with &{}.

>> n = 42
write("The answer is &{n}")
write("2 + 2 = &{2 + 2}")

Concatenate with +:

>> s = "Hello" + ", " + "World"

Comments use # (single-line) or #/ ... /# (multi-line).

Control Flow

if / elif / else

if x > 0 +/
    write("positive")
ECB elif x == 0 +/
    write("zero")
ECB else +/
    write("negative")
ECB

while

>> i = 0
while i < 5 +/
    write(i)
    i = i + 1
ECB

for

for item in [1, 2, 3] +/
    write(item)
ECB

for i in range(10) +/
    write(i)
ECB

Functions

bladeFN greet(name#str) +/
    write("Hello, &{name}!")
ECB

greet("Alon")

Functions are values — pass them around, return them, or define them inline:

>> double = bladeFN(x) +/ return x * 2 ECB

Closures capture the enclosing scope at definition time:

bladeFN make_adder(n) +/
    return bladeFN(x) +/
        return x + n
    ECB
ECB

>> add5 = make_adder(5)
write(add5(3))   # 8

Classes

Use bladeGRP to define a class. The instance reference is blade (not self).

bladeGRP Point +/
    bladeFN init(x#int, y#int) +/
        blade.x = x
        blade.y = y
    ECB
    bladeFN to_str() +/
        return "(&{blade.x}, &{blade.y})"
    ECB
ECB

>> p = Point(3, 4)
write(p.to_str())

Type Annotations

Add #type after a parameter name to enforce the type at call time.

AnnotationType checked
name#strstring
count#intinteger (whole number)
value#floatfloat or int
flag#boolboolean

Passing the wrong type raises AxonTypeError.

Color Literals

Color literals are values of the form -*name*-. They produce ANSI escape codes when printed.

LiteralColorLiteralColor
-*red*-Red-*blue*-Blue
-*green*-Green-*cyan*-Cyan
-*yellow*-Yellow-*magenta*-Magenta
-*white*-White-*black*-Black
-*reset*-Reset (no color)
write(-*red*- + "Error!" + -*reset*-)

Error Handling

try +/
    >> x = 10 / 0
ECB catch err +/
    write("Caught: &{err.message}")
ECB

The err variable is a dict with type, message, and line fields.

Raise your own errors with:

raise "something went wrong"

Modules

uselib -math-
write(math.sqrt(16))   # 4.0

uselib -string-
write(string.upper("hello"))   # HELLO

You can also import local files: uselib -./myfile- loads myfile.axb.
