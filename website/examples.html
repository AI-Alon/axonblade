Examples

Working programs covering every core feature of AxonBlade — from hello world to terminal games.

Hello World

Colors, string interpolation, and a simple typed function.

# AxonBlade — Hello World >> colors = [-*red*-, -*green*-, -*yellow*-, -*cyan*-, -*magenta*-, -*blue*-] >> names = ["red", "green", "yellow", "cyan", "magenta", "blue"] # Print each color >> i = 0 while i < len(colors) +/ write(colors[i] + " This is &{names[i]} " + -*reset*-) i = i + 1 ECB write("") write(-*cyan*- + "Hello, AxonBlade!" + -*reset*-) # Typed function bladeFN greet(name#str, n#int) +/ >> i = 0 while i < n +/ write(-*green*- + "Hi, &{name}!" + -*reset*-) i = i + 1 ECB ECB greet("World", 3)
Concepts: variables, color literals, string interpolation, while loop, typed function

Fibonacci

Recursive function, closure generator, and higher-order functions.

# Recursive Fibonacci bladeFN fib(n#int) +/ if n <= 1 +/ return n ECB return fib(n - 1) + fib(n - 2) ECB write(-*cyan*- + "Recursive Fibonacci:" + -*reset*-) for i in range(10) +/ write("fib(&{i}) = &{fib(i)}") ECB # Closure-based generator bladeFN fib_gen() +/ >> a = 0 >> b = 1 return bladeFN() +/ >> v = a >> next = a + b a = b b = next return v ECB ECB write("") write(-*green*- + "Generator (first 10):" + -*reset*-) >> gen = fib_gen() for _ in range(10) +/ write(gen()) ECB
Concepts: recursion, closures, for loop, higher-order functions

Closures

Counter factories, adder generators, and pipeline composition.

# Counter factory bladeFN make_counter(start) +/ >> n = start return bladeFN() +/ n = n + 1 return n ECB ECB >> c1 = make_counter(0) >> c2 = make_counter(100) write(c1()) # 1 write(c1()) # 2 write(c2()) # 101 — independent state # Adder factory + pipeline bladeFN make_adder(n) +/ return bladeFN(x) +/ return x + n ECB ECB bladeFN double(x) +/ return x * 2 ECB >> add10 = make_adder(10) >> result = 5 |> double() |> add10() write(result) # 20 # map_list helper bladeFN map_list(lst, fn) +/ >> out = [] for x in lst +/ out.append(fn(x)) ECB return out ECB >> nums = [1, 2, 3, 4, 5] write(map_list(nums, double)) # [2, 4, 6, 8, 10]
Concepts: closures, factory functions, pipeline operator, list methods

Classes

OOP with bladeGRP — Animal health system, Point class, try/catch.

bladeGRP Animal +/ bladeFN init(name#str, hp#int) +/ blade.name = name blade.hp = hp blade.alive = true ECB bladeFN damage(amount#int) +/ blade.hp = blade.hp - amount if blade.hp <= 0 +/ blade.alive = false write(-*red*- + "&{blade.name} is defeated!" + -*reset*-) ECB else +/ write("&{blade.name}: &{blade.hp} hp") ECB ECB ECB >> wolf = Animal("Wolf", 50) wolf.damage(20) wolf.damage(40) # try/catch write("") try +/ >> bad = 10 / 0 ECB catch e +/ write(-*yellow*- + "Caught: &{e.message}" + -*reset*-) ECB
Concepts: bladeGRP classes, blade keyword, type annotations, if/else, try/catch

Snake Game

Playable Snake in the terminal — grid, game loop, keyboard input, borders.

# Snake — run with: ablade run examples/snake.axb # Controls: WASD to move, Q to quit # (truncated for display — see full source in examples/snake.axb) >> W = 30 >> H = 18 >> g = grid(W, H) # Snake body as list of [x, y] tiles >> snake = [[5, 5], [4, 5], [3, 5]] >> dir_x = 1 >> dir_y = 0 # Key bindings g.on_key("w", bladeFN() +/ next_dx = 0 next_dy = -1 ECB) g.on_key("s", bladeFN() +/ next_dx = 0 next_dy = 1 ECB) g.on_key("a", bladeFN() +/ next_dx = -1 next_dy = 0 ECB) g.on_key("d", bladeFN() +/ next_dx = 1 next_dy = 0 ECB) g.on_key("q", bladeFN() +/ g.stop() ECB) g.loop(bladeFN() +/ # move, draw, check collisions ... ECB, 10)
Concepts: grid, on_key, loop, list methods, game loop pattern
Run locally: ablade run examples/snake.axb

Conway's Game of Life

Interactive editor + live simulation. Draw cells, then press Enter to run.

# Game of Life — interactive editor + simulation # Controls: arrows = move cursor, space = toggle cell # enter = start sim, p = pause, q = quit >> W = 40 >> H = 22 >> g = grid(W, H) >> cells = [] # alive = list of [x, y] >> paused = true >> cx = 20 >> cy = 11 bladeFN count_neighbors(x, y) +/ >> count = 0 for nb in cells +/ >> dx = nb[0] - x >> dy = nb[1] - y if (dx * dx + dy * dy <= 2) -a -n (dx == 0 -a dy == 0) +/ count = count + 1 ECB ECB return count ECB g.loop(bladeFN() +/ # draw grid, advance simulation ... ECB, 10)
Concepts: grid, interactive editor, simulation, list operations, -a / -n operators
Run locally: ablade run examples/life.axb
