Examples

Working programs covering every core feature of AxonBlade — from hello world to terminal games.

Hello World

Colors, string interpolation, and a simple typed function.

# hello.axb — Welcome to AxonBlade # Demonstrates: variables, string interpolation, colors, loops, functions >> name = "AxonBlade" >> version = 1 write(-*cyan*- + "Welcome to &{name} v&{version}" + -*reset*-) write("") # Colors as first-class values >> colors = [-*red*-, -*yellow*-, -*green*-, -*cyan*-, -*blue*-, -*magenta*-] >> labels = ["red", "yellow", "green", "cyan", "blue", "magenta"] >> i = 0 while i < len(colors) +/ write(colors~i~ + " " + labels~i~ + " " + -*reset*-) i = i + 1 ECB write("") # String interpolation with expressions >> a = 6 >> b = 7 write("&{a} * &{b} = &{a * b}") # Function definition bladeFN greet(who#str) +/ return -*green*- + "Hello, &{who}!" + -*reset*- ECB write(greet("world")) write(greet(name))
Concepts: variables, color literals, string interpolation, while loop, typed function

Fibonacci

Recursive function, closure generator, and higher-order functions.

# fibonacci.axb — Fibonacci sequence # Demonstrates: recursion, closures, first-class functions, pipeline # Recursive Fibonacci bladeFN fib(n#int) +/ if n <= 1 +/ return n ECB return fib(n - 1) + fib(n - 2) ECB write(-*cyan*- + "Recursive Fibonacci:" + -*reset*-) >> i = 0 while i < 10 +/ write(" fib(&{i}) = &{fib(i)}") i = i + 1 ECB write("") # Fibonacci using a closure-based memoizing generator bladeFN make_fib_gen() +/ >> a = 0 >> b = 1 bladeFN next() +/ >> val = a >> tmp = b b = a + b a = tmp return val ECB return next ECB write(-*cyan*- + "Generator Fibonacci:" + -*reset*-) >> gen = make_fib_gen() >> j = 0 while j < 10 +/ write(" &{gen()}") j = j + 1 ECB write("") # First-class functions: apply a transform to each fib number bladeFN apply(fn#fn, n#int) +/ return fn(n) ECB bladeFN square(x#int) +/ return x * x ECB write(-*cyan*- + "Squared Fibonacci:" + -*reset*-) >> k = 0 while k < 8 +/ write(" fib(&{k})^2 = &{apply(square, fib(k))}") k = k + 1 ECB
Concepts: recursion, closures, for loop, higher-order functions

Closures

Counter factories, adder generators, and pipeline composition.

# closures.axb — Closures and higher-order functions # Demonstrates: closures, captured state, returning functions, pipelines # 1. Counter — closure captures and mutates its own variable bladeFN make_counter(start#int) +/ >> count = start bladeFN increment() +/ count = count + 1 return count ECB bladeFN reset() +/ count = start ECB bladeFN get() +/ return count ECB return [increment, reset, get] ECB write(-*cyan*- + "Counter:" + -*reset*-) >> fns = make_counter(0) >> inc = fns~0~ >> rst = fns~1~ >> get = fns~2~ write(" start: &{get()}") write(" after 3 increments: &{inc()} &{inc()} &{inc()}") write(" reset") rst() write(" after reset: &{get()}") write("") # 2. Adder factory — each closure captures a different value bladeFN make_adder(n#int) +/ bladeFN add(x#int) +/ return x + n ECB return add ECB write(-*cyan*- + "Adder factory:" + -*reset*-) >> add5 = make_adder(5) >> add10 = make_adder(10) >> add100 = make_adder(100) write(" add5(3) = &{add5(3)}") write(" add10(3) = &{add10(3)}") write(" add100(3) = &{add100(3)}") write("") # 3. Pipeline: chain functions with |> bladeFN double(x#int) +/ return x * 2 ECB bladeFN inc(x#int) +/ return x + 1 ECB bladeFN square(x#int) +/ return x * x ECB write(-*cyan*- + "Pipeline 3 |> double |> inc |> square:" + -*reset*-) >> result = 3 |> double() |> inc() |> square() write(" result = &{result}") write("") # 4. Higher-order map bladeFN map_list(lst#list, fn#fn) +/ >> out = [] for item in lst +/ out.append(fn(item)) ECB return out ECB >> nums = [1, 2, 3, 4, 5] write(-*cyan*- + "map double over &{nums}:" + -*reset*-) >> doubled = map_list(nums, double) for n in doubled +/ write(" &{n}") ECB
Concepts: closures, factory functions, pipeline operator, list methods

Classes

OOP with bladeGRP — Animal health system, Point class, try/catch.

# classes.axb — bladeGRP (classes) and instances # Demonstrates: bladeGRP definition, init, methods, fields, try/catch, error handling bladeGRP Animal +/ bladeFN init(blade, name#str, sound#str, hp#int) +/ blade.name = name blade.sound = sound blade.hp = hp blade.alive = true ECB bladeFN speak(blade) +/ if blade.alive +/ write("&{blade.name} says: &{blade.sound}!") ECB else +/ write("&{blade.name} is silent.") ECB ECB bladeFN take_damage(blade, amount#int) +/ blade.hp = blade.hp - amount if blade.hp <= 0 +/ blade.hp = 0 blade.alive = false write(-*red*- + "&{blade.name} has been defeated!" + -*reset*-) ECB else +/ write("&{blade.name} takes &{amount} damage. HP: &{blade.hp}") ECB ECB bladeFN describe(blade) +/ >> status = "alive" if -n blade.alive +/ status = "defeated" ECB return "&{blade.name} [HP: &{blade.hp}] (&{status})" ECB ECB bladeGRP Point +/ bladeFN init(blade, x#int, y#int) +/ blade.x = x blade.y = y ECB bladeFN distance(blade, other) +/ >> dx = blade.x - other.x >> dy = blade.y - other.y return dx * dx + dy * dy ECB bladeFN to_str(blade) +/ return "(&{blade.x}, &{blade.y})" ECB ECB # -- Animals -- write(-*cyan*- + "=== Animals ===" + -*reset*-) >> dog = Animal("Rex", "woof", 30) >> cat = Animal("Luna", "meow", 20) dog.speak() cat.speak() write(dog.describe()) write(cat.describe()) write("") dog.take_damage(10) dog.take_damage(15) dog.take_damage(10) dog.speak() write(dog.describe()) write("") # -- Points -- write(-*cyan*- + "=== Points ===" + -*reset*-) >> p1 = Point(0, 0) >> p2 = Point(3, 4) write("p1 = &{p1.to_str()}") write("p2 = &{p2.to_str()}") write("squared distance = &{p1.distance(p2)}") write("") # -- Error handling -- write(-*cyan*- + "=== Error Handling ===" + -*reset*-) bladeFN safe_divide(a#int, b#int) +/ if b == 0 +/ raise "Division by zero" ECB return a / b ECB try +/ write("10 / 2 = &{safe_divide(10, 2)}") write("5 / 0 = &{safe_divide(5, 0)}") ECB catch e +/ write(-*red*- + "Caught: &{e}" + -*reset*-) ECB
Concepts: bladeGRP classes, blade keyword, type annotations, if/else, try/catch

Snake Game

Playable Snake in the terminal — grid, game loop, keyboard input, borders.

# Snake — run with: ablade run examples/snake.axb # Controls: w/a/s/d to move, q to quit >> W = 24 >> H = 16 >> g = grid(W, H) >> snake = [{x: 12, y: 8}, {x: 11, y: 8}, {x: 10, y: 8}] >> dir_x = 1 >> dir_y = 0 >> next_dx = 1 >> next_dy = 0 >> food_x = 5 >> food_y = 4 >> score = 0 >> alive = true >> tick = 0 bladeFN is_on_snake(px, py) +/ >> i = 0 while i < len(snake) +/ >> seg = snake~i~ if seg~"x"~ == px -a seg~"y"~ == py +/ return true ECB i = i + 1 ECB return false ECB bladeFN draw() +/ g.fill(-*black*-) >> bx = 0 while bx < W +/ g.set(bx, 0, -*white*-) g.set(bx, H-1, -*white*-) bx=bx+1 ECB g.set(food_x, food_y, -*red*-) g.set_char(food_x, food_y, "*") >> i = 1 while i < len(snake) +/ >> s = snake~i~ g.set(s~"x"~, s~"y"~, -*green*-) i=i+1 ECB >> head = snake~0~ g.set(head~"x"~, head~"y"~, -*cyan*-) g.set_char(head~"x"~, head~"y"~, "@") ECB bladeFN update() +/ if -n alive +/ return null ECB tick = tick + 1 dir_x = next_dx dir_y = next_dy >> head = snake~0~ >> nx = head~"x"~ + dir_x >> ny = head~"y"~ + dir_y if nx <= 0 -o nx >= W-1 -o ny <= 0 -o ny >= H-1 +/ alive = false g.stop() return null ECB if is_on_snake(nx, ny) +/ alive=false g.stop() return null ECB snake.insert(0, {x: nx, y: ny}) if nx == food_x -a ny == food_y +/ score = score + 1 food_x = (food_x * 7 + tick * 3 + 1) % (W-2) + 1 food_y = (food_y * 5 + tick * 2 + 1) % (H-2) + 1 ECB else +/ snake.pop() ECB draw() ECB g.on_key("w", bladeFN() +/ if dir_y==0 +/ next_dx=0 next_dy=-1 ECB ECB) g.on_key("s", bladeFN() +/ if dir_y==0 +/ next_dx=0 next_dy=1 ECB ECB) g.on_key("a", bladeFN() +/ if dir_x==0 +/ next_dx=-1 next_dy=0 ECB ECB) g.on_key("d", bladeFN() +/ if dir_x==0 +/ next_dx=1 next_dy=0 ECB ECB) g.on_key("q", bladeFN() +/ g.stop() ECB) draw() g.loop(update, 8)
Concepts: grid, on_key, loop, list methods, game loop pattern
Run locally: ablade run examples/snake.axb

Conway's Game of Life

Interactive editor + live simulation. Draw cells, then press Enter to run.

# Game of Life — interactive editor + simulation # EDIT: w/a/s/d move cursor, space toggle, enter start # RUN: p pause/resume, q quit >> W = 30 >> H = 20 >> g = grid(W, H) >> paused = true >> cx = 15 >> cy = 10 bladeFN make_board() +/ >> b = [] >> y = 0 while y < H +/ >> row = [] >> x = 0 while x < W +/ row.append(0) x=x+1 ECB b.append(row) y=y+1 ECB return b ECB >> board = make_board() bladeFN count_neighbors(brd, px, py) +/ >> total = 0 >> dy = -1 while dy <= 1 +/ >> dx = -1 while dx <= 1 +/ if -n (dx==0 -a dy==0) +/ >> nx=px+dx >> ny=py+dy if nx>=0 -a nx<W -a ny>=0 -a ny<H +/ total = total + brd~ny~~nx~ ECB ECB dx=dx+1 ECB dy=dy+1 ECB return total ECB bladeFN step() +/ >> next = make_board() >> y=0 while y<H +/ >> x=0 while x<W +/ >> n=count_neighbors(board,x,y) if board~y~~x~==1 -a (n==2 -o n==3) +/ next~y~~x~=1 ECB elif board~y~~x~==0 -a n==3 +/ next~y~~x~=1 ECB x=x+1 ECB y=y+1 ECB board = next ECB bladeFN update() +/ if -n paused +/ step() ECB >> y=0 while y<H +/ >> x=0 while x<W +/ if board~y~~x~==1 +/ g.set(x,y,-*green*-) ECB else +/ g.set(x,y,-*black*-) ECB x=x+1 ECB y=y+1 ECB ECB g.on_key("w", bladeFN() +/ if cy>0 +/ cy=cy-1 ECB ECB) g.on_key("s", bladeFN() +/ if cy<H-1 +/ cy=cy+1 ECB ECB) g.on_key("a", bladeFN() +/ if cx>0 +/ cx=cx-1 ECB ECB) g.on_key("d", bladeFN() +/ if cx<W-1 +/ cx=cx+1 ECB ECB) g.on_key(" ", bladeFN() +/ if paused +/ if board~cy~~cx~==1 +/ board~cy~~cx~=0 ECB else +/ board~cy~~cx~=1 ECB ECB ECB) g.on_key("\r", bladeFN() +/ paused=false ECB) g.on_key("p", bladeFN() +/ if paused +/ paused=false ECB else +/ paused=true ECB ECB) g.on_key("q", bladeFN() +/ g.stop() ECB) g.loop(update, 10)
Concepts: grid, interactive editor, simulation, list operations, -a / -n operators
Run locally: ablade run examples/life.axb
